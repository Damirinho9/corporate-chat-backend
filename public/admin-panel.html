📚 Документация по API

🤖 Bot Management API

Создание бота:

POST /api/bots
Authorization: Bearer <admin_token>

{
  "name": "My Bot",
  "username": "my_bot",
  "description": "Helpful bot"
}

Добавление прав:

POST /api/bots/:id/permissions
Authorization: Bearer <admin_token>

{
  "permission_type": "send_messages",
  "resource_type": "chat",
  "resource_id": 5
}

🔗 Webhooks API

Создание вебхука:

POST /api/webhooks
Authorization: Bearer <admin_token>

{
  "bot_id": 1,
  "name": "My Webhook",
  "url": "https://example.com/webhook",
  "events": ["message.created", "user.joined"]
}

🤖 Bot API (для использования ботами)

Отправка сообщения:

POST /api/bot-api/messages
X-Bot-Token: bot_abc123...

{
  "chat_id": 5,
  "content": "Hello from bot!"
}

🔐 HMAC Signature Verification

Все webhook запросы включают заголовок X-Webhook-Signature с HMAC-SHA256 подписью.

const crypto = require('crypto');

function verifySignature(secret, payload, signature) {
  const hmac = crypto.createHmac('sha256', secret);
  hmac.update(JSON.stringify(payload));
  return hmac.digest('hex') === signature;
}

📋 Доступные события

  • ✅ message.created - Новое сообщение
  • ✅ message.updated - Сообщение изменено
  • ✅ message.deleted - Сообщение удалено
  • ✅ user.joined - Пользователь присоединился
  • ✅ user.left - Пользователь вышел
  • ✅ chat.created - Чат создан
  • ✅ chat.updated - Чат обновлен
  • ✅ call.started - Звонок начался
  • ✅ call.ended - Звонок завершен
  • ✅ file.uploaded - Файл загружен
  • ✅ reaction.added - Реакция добавлена
